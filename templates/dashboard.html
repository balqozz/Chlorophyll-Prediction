Lettuce Leaf Chlorophyll Prediction System

An innovative digital platform integrating RGB color features and Infrared (IR) intensity data, processed using the Random Forest ensemble algorithm.

Key Platform Features:
  • Single-File Batch Analysis: Processes one multispectral dataset file (.xlsx) per run, supporting multiple lettuce samples within that single file.
  • Multisensor Intelligence: Automatically computes the Excess Green (ExG) vegetation index from RGB, combined with active Infrared (IR) sensor readings.
  • Leave-One-Out Cross-Validation: Rigorously validates prediction stability of the Random Forest Regressor across all 16 lettuce samples.
  • Arnon's Reference Ground Truth: Chlorophyll a, b, and Total Chlorophyll are each independently predicted, then benchmarked against real UV-Vis spectrophotometer lab measurements calculated via Arnon's (1949) equations.
